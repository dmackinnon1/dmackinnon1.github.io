mathrecreation blog

A place for some of my mathematical recreations.

polygrid

Generates examples of polynomial division using the grid method. Source in the polygrid repo. Information about the method can be found on various posts linked to this one.

kixote, hidato & KnightTourist

Interactive knight's and king's tour puzzles. Source in the kixote repo. More info on kixote here and here, and on hidato here. The KnightTourist page lets you find your own tour while providing you with some clues for where to move.

quiltic

A page for playing around with some celtic plait motifs. Source in the quiltic repo. Information on the inspiration for this can be found here.

cplxfrac

Some Mandelbrot and Julia set fractals. Source in the cplxfrac repo. A post about plotting these can be found here.

simple R examples

Some simple R programming examples. Includes some scripts that generate Chaladni figures (see here), and a Monty Hall simulation (see here), for example.
